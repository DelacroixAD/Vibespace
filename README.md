# VibeSpace 🌿

VibeSpace is a productivity-focused web app designed to help users plan, focus, and stay motivated.

## Features
- 📝 To-Do List
- 📅 Daily Planner
- 💬 Motivation
- ⏱ Pomodoro Timer
- 🎯 Daily Goals

## Tech Stack
- HTML
- CSS
- JavaScript

## Current Status
- Home page with interactive feature cards
- Section-based navigation with open/close flow

## Future Plans
- User authentication
- Data persistence
- Backend integration (PHP / DB)
